## 1. Règle éditoriale principale

Le landing général doit vendre la marque KORYXA Formation et orienter vers les formations, sans promettre des produits non disponibles.

Règle:

```txt
La page générale présente l'écosystème.
Les landings dédiées vendent chaque formation.
KORYXA Admin gouverne les produits et accès.
Partner Portal distribue les liens et ressources partenaires.
```

Ton recommandé:

```txt
professionnel
clair
premium
concret
orienté résultats
sans hype IA inutile
```

## 12. FAQ portail général

### Question 1

```txt
Quelle formation est disponible aujourd'hui ?
```

Réponse:

```txt
La formation Analyse de données avec Python est le parcours principal actuellement disponible. Elle couvre Python, Pandas, NumPy, la visualisation, l'analyse exploratoire et un projet final portfolio.
```

### Question 2

```txt
Les formations LLM/RAG, Assistant IA et Automatisation sont-elles déjà ouvertes ?
```

Réponse:

```txt
Non. Ces parcours sont en préparation ou en roadmap. Ils peuvent être présentés sur le portail, mais ils ne doivent pas être considérés comme ouverts à l'achat tant que leur offre, leur landing dédiée et leur tunnel d'accès ne sont pas validés.
```

### Question 3

```txt
Est-ce adapté aux débutants ?
```

Réponse:

```txt
Oui, selon le parcours. La formation Python Data Analyst démarre progressivement et accompagne l'apprenant vers des notebooks, des exercices et un projet final.
```

### Question 4

```txt
Est-ce seulement une plateforme de vidéos ?
```

Réponse:

```txt
Non. L'objectif de KORYXA Formation est de guider l'apprenant vers une pratique réelle, avec modules, exercices, ressources, projet final et preuve de compétence.
```

### Question 5

```txt
Comment fonctionne l'accès aux formations ?
```

Réponse:

```txt
L'accès dépend du parcours et du tunnel actif. Certaines entrées passent par la page d'accès Formation, et les accès partenaires peuvent être attribués via l'écosystème KORYXA Admin et Partner Portal.
```

### Question 6

```txt
Les partenaires ont-ils un espace dédié ?
```

Réponse:

```txt
Oui. Les partenaires KORYXA utilisent un espace dédié pour suivre leurs liens, leurs prospects, leurs ressources marketing et leurs commissions sur les produits publiés.
```

### Question 7

```txt
Le certificat est-il inclus ?
```

Réponse:

```txt
Le certificat fait partie du parcours Python Data Analyst après complétion des étapes prévues. Les règles de certificat peuvent varier selon les futures formations.
```

## 17. Règles anti-erreur pour l'intégration

```txt
1. Ne pas écrire "Acheter" sur LLM/RAG tant que le produit n'est pas validé.
2. Ne pas afficher de prix pour les formations roadmap.
3. Ne pas promettre un accès immédiat pour les formations en préparation.
4. Ne pas exposer KORYXA Admin comme destination publique.
5. Ne pas remplacer le Partner Portal par une section partenaire publique.
6. Ne pas créer de CTA sans route validée.
7. Ne pas surcharger le hero avec trop de texte.
8. Ne pas utiliser la page générale comme page de vente détaillée d'une formation spécifique.
```
