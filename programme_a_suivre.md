Maîtriser l'analyse de données avec Python, de l'installation jusqu'à un projet professionnel complet, pour tous niveaux — même sans expérience préalable.

## 📚 MODULE 0 — Introduction & Installation *(Mise en place)*

- Qu'est-ce que l'analyse de données et pourquoi Python ?
- Installation de Python + Anaconda / Jupyter Notebook
- Présentation de l'environnement Jupyter
- Votre premier notebook : écrire et exécuter du code
- Bonnes pratiques d'organisation d'un projet data

## 📚 MODULE 1 — Les Bases de Python pour la Data *(Fondations)*

- Variables, types de données (int, float, str, bool)
- Listes, tuples, dictionnaires et ensembles
- Conditions (if/elif/else) et boucles (for/while)
- Fonctions : définir, appeler, retourner
- Introduction aux bibliothèques : `import` et `pip install`

## 📚 MODULE 2 — NumPy : Calcul Numérique *(Puissance mathématique)*

- Introduction à NumPy et les arrays
- Opérations mathématiques vectorisées
- Indexation, slicing et filtrage
- Fonctions statistiques de base (mean, std, min, max)
- Comparaison numpy vs listes Python (vitesse & efficacité)

## 📚 MODULE 3 — Pandas : Manipulation de Données *(Le cœur de la Data)*

- Introduction aux DataFrames et Series

- Exploration : `.head()`, `.info()`, `.describe()`

- Tri, regroupement et agrégation (`groupby`)

- Exportation des résultats

## 📚 MODULE 4 — Nettoyage de Données *(Data Cleaning)*

- Identifier et gérer les valeurs manquantes
- Supprimer et corriger les doublons
- Standardisation des formats (dates, textes, nombres)
- Détection et traitement des valeurs aberrantes (outliers)
- Renommer, réorganiser et restructurer un dataset
- Pipeline de nettoyage réutilisable

---

## 📚 MODULE 5 — Visualisation de Données *(Raconter avec des graphiques)*

- Introduction à Matplotlib : courbes, barres, histogrammes
- Seaborn : graphiques statistiques avancés et élégants
- Personnalisation : couleurs, titres, légendes, tailles
- Visualisations clés : heatmap, boxplot, pairplot, scatter
- Sauvegarder et exporter vos graphiques en haute qualité

## 📚 MODULE 6 — Analyse Exploratoire (EDA) *(Trouver les insights)*

- C'est quoi une EDA et comment la structurer ?
- Analyse univariée, bivariée et multivariée
- Corrélations et relations entre variables
- Questions métier → insights data
- Rédiger et présenter vos conclusions
